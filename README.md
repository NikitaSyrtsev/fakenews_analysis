# fakenews_detection

Все данные, которые были использованы в проект загружены в виде архивов, так как они имеют большой размер. Для работы с ними их необходимо распаковать. 

# Использование
В файле fake_news_detector_app содержатся сохраненные файлы обученной модели и app.py, который запускает приложение для тестирования модели машинного обучения по команде python app.py в консоли. Мы можете вставить текст новости и получить метку модели с вероятностью о фейковости новости.

Для использования необходимо скачать библиотеки по команде: pip install -r requirements.txt
